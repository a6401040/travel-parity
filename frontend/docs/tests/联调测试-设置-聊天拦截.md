# 联调测试文档：设置-聊天页面未登录拦截

## 1. 目标
- 未登录用户访问 `/chat` 时自动拦截并提示“请先登录”，跳转到 `/auth/login`。

## 2. 步骤
- 清除本地登录状态（退出登录或清理浏览器 Cookie）。
- 在浏览器直接访问 `http://localhost:5173/chat`。

## 3. 预期
- 弹出提示：`请先登录`。
- 跳转到登录页：`http://localhost:5173/auth/login`。

## 4. 实施要点
- 前端路由对 `/chat` 设置 `meta.requiresAuth=true`。
- 全局前置守卫在未认证时拦截并提示。
- 后端无须变更（保持会话校验在受保护接口）。

## 5. 结果
- 验证通过：未登录无法进入对话页面，体验一致且提示明确。
